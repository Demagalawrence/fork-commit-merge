# Markdown Guide

## Headers

Use `#` for headers. The number of `#` symbols indicates the level.

# H1
## H2
### H3
#### H4
##### H5
###### H6

## Lists

**Unordered list:**
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

**Ordered list:**
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2

## Links

Format: `[link text](URL)`

Example: [GitHub](https://github.com)

## Images

Format: `![alt text](image_url)`

Example:

![Markdown Logo](https://markdown-here.com/img/icon256.png)

## Code Blocks

Inline code: Use backticks, e.g. `code`.

Multiline code block:

```
function hello() {
  console.log('Hello, world!');
}
```

## Tables

| Syntax | Description |
|--------|-------------|
| Header | Title       |
| Cell   | Data        |

Align columns:

| Left   | Center | Right  |
|:-------|:------:|-------:|
